# UCC INGO — BSCS Information Board

The official website of the University of Caloocan City Computer Science Council. Built with Next.js 15, Sanity CMS v3, and Google Gemini AI.

## Quick Start

```bash
npm install
cd studio && npm install && cd ..
npm run dev
```

- Front-end: http://localhost:3000
- CMS: http://localhost:3333
