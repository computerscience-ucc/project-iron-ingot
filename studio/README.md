# UCC INGO — Sanity Studio v3

Content management backend for the UCC INGO website.

## Commands

```bash
sanity dev       # Start dev server (port 3333)
sanity build     # Build static files
sanity deploy    # Deploy to Sanity hosting
```

All schemas use `defineType()` from the `sanity` package. See [AGENTS.md](../AGENTS.md) for the full maintenance guide.
